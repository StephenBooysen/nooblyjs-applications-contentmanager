Quick Start Guide

Integrate Noobly Applications into your project in minutes

1. Installation

Clone the Noobly Applications repository:

git clone https://github.com/nooblyjs/applications.git

2. Configure Applications

Select and configure the applications you need:

npm install npm start

3. Start Building

Access applications at localhost:3001:

// Visit /applications/ // Or integrate individual modules
